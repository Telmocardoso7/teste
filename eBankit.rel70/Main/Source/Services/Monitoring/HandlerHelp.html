Probes Handler Helper

Usage

Structure

<monitoring-address>/.monitor?<querystring>

Tests
1. Ping Test

Test=1

Target=<target-address>

Example: http://example.local/.monitor?test=1&target=example.local/machine

2. Database Connection Test

Test=2

Db=<connection string name>

Example: http://example.local/.monitor?test=2&db=ExampleConnectionString

3. Endpoint Invocation Test

Test=3

Headers=<header1-name:header1-value|header2-name:header2-value|…>

EndPoint=<endpoint>

Example: http://example.local/.monitor?test=3&headers=ITSAPP-DEVICE:ANDROIDTABLET|ITSAPP-LANG:pt-PT|ITSAPP-VER:1.2.1|ITSAPP-SO:21&endPoint=http://example.local/public/login
